tail -f app.log | claude -p "监控日志并发现异常时通知我"

⚡ 斜杠命令：高效操作的快捷键

斜杠命令是Claude Code中的"魔法咒语"，让你能够快速执行特定操作，极大提升工作效率。

命令 分类 功能描述 使用场景
/clear 对话管理 清空当前对话历史，释放上下文空间 开始新任务时
/compact 对话管理 智能压缩对话历史，保留关键信息 长会话优化
/init 项目管理 分析代码库并生成CLAUDE.md文件 项目初始化
/cost 成本管理 查看当前会话Token使用量和费用 成本监控
/model 配置管理 切换使用的Claude模型 性能调优
/mcp 扩展功能 管理模型上下文协议服务器 能力扩展

MCP配置示例
# 添加PostgreSQL MCP服务器
claude mcp add pg_server /path/to/postgres-mcp-server \
  --connection-string "postgresql://user:pass@host:port/db"

# 使用MCP查询数据库
> 请描述 @pg_server:schema://users 表的结构

# 查询业务数据
> 使用 @pg_server:db://sales_db 查询上个季度销售额最高的三个产品

# GitHub集成示例
